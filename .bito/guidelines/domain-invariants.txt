ContentfulBundle Domain Invariants

1. EXACTLY ONE DEFAULT CLIENT. ContentfulExtension throws LogicException at container compile time if the number of clients with `default: true` is not exactly 1. Never add a second client configuration without ensuring exactly one is marked default.

2. DELIVERY/PREVIEW API ONLY. This bundle wraps contentful/contentful (CDA SDK). Never introduce CMA (Content Management API) operations — that is out of scope.

3. PROFILER CONTROLLER IS CONDITIONAL. ProfilerController is only registered when both `profiler` and `twig` services exist in the container (checked by ProfilerControllerPass). Never assume it is always available. Never hardcode a reference to `contentful.profiler_controller` without checking.

4. SYMFONY INTEGRATION TELEMETRY. ClientFactory always calls `$client->useIntegration(new SymfonyIntegration())`. Do not remove or skip this call — it's SDK telemetry for framework adoption tracking.

5. CACHE IS OPT-IN. Neither resource cache nor query cache is active by default. Do not add default cache pool wiring. Users must explicitly configure `options.cache.pool` or `options.query_cache.pool`.

6. QUERY CACHE HAS NO INVALIDATION. The query cache TTL defaults to 60 seconds with no invalidation mechanism. Never raise this default without documenting the stale content trade-off explicitly.

7. PSR INTERFACES FOR EXTENSIBILITY. Logger and cache pool are injected via PSR-3/PSR-6 interface references, not concrete Symfony types. Keep these as PSR interfaces — never replace with Symfony-specific types.

8. MASTER IS THE DEFAULT BRANCH. This repo uses `master`, not `main`. CI, CODEOWNERS, and all release processes target `master`.

9. PHPSTAN LEVEL 4. Do not lower the static analysis level. The two existing `@phpstan-ignore-line` comments in Configuration.php are intentional workarounds for a known PHPStan limitation with chained node builders.

10. PHP VERSION MATRIX. CI tests PHP 8.0–8.4. The `require` in composer.json includes `^7.4`. Do not drop PHP version support without a major version bump and CHANGELOG entry.
